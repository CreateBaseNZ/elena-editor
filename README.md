# Elena Editor

This is a project to provide various types of editors to front end developers.

**Current Progress**

- Flow Chart Editor - React

**Coming Features**

- Monaco Editor

## Installation

```js
npm install elena-editor
```

or

```js
yarn add elena-editor
```
